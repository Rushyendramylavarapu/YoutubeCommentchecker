# YouTube Comments Sentiment Analysis

A Python-based tool to analyze sentiment of YouTube comments using the YouTube Data API and VADER sentiment analysis.

## Features
- Fetch comments from YouTube videos.
- Clean and preprocess text.
- Analyze sentiment (Positive, Negative, Neutral).
- Visualize results with Pie Charts, Bar Graphs, and Word Clouds.
- Export results to CSV.

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python -m streamlit run app.py
   ```

## API Key
You will need a YouTube Data API v3 key from the [Google Cloud Console](https://console.cloud.google.com/).
